position : static:

An element with position:static; is not positioned in any special way.It is always positioned according to the normal flow of the page. By default all the elements are in position static.

The div element has position: static;

position : relative;

An element with position: relative; is positioned rel;ative to its normal position

This div element has position : relative;

position : fixed;

An element with postion:fixed; is positioned relate to the viewport, which means it always stays in the same place even if the page is scrolled

This div element is position:fixed;

position:absolute;

An element with position : absolute; i positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).

This div element has position: relative;
This div element is position: absolute;
